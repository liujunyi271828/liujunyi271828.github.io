## CSS基础

### CSS简介

1. CSS = 层叠样式表(Cascading Style Sheets)
2. CSS主要用于定义HTML内容在浏览器内的显示样式, 如字体大小、颜色、字体加粗等.
3. 优势: 极大地提高了工作效率.


### CSS代码语法

selector {
    property: value;
}

例子:

```css
h1 {
    color: blue;
    font-size: 20px;
}
```
