举例: 


```php
<?php

/**
 * 匹配网页中的<img/>元素.
 */

$path = 'https://liujunyi271828.github.io/2019-04-08/ling-yan-ge-xu/';
$htmlString = file_get_contents($path);
preg_match_all('#<img[^>]*>#i', $htmlString, $matches);
var_dump($matches);
foreach ($matches[0] as $match) {
    echo $match . "<br/>";
}
```
